# 🐍 Snake Game

A fun and smooth **Snake Game** developed with C# Windows Forms.  
Includes a modern UI, score system, speed settings, color customization, and leaderboard.

![Snake Game](https://raw.githubusercontent.com/elcanaliakberli/SnakeGame/refs/heads/main/snake-game.png)

---

## ✨ Features

- 🎮 Modern user interface
- 🐍 Advanced snake design (eyes, tongue, shine effects)
- 🍎 Styled food graphics
- ⚡ 3 Speed Modes
  - Slow
  - Medium
  - Fast
- 🎨 Change snake color
- 🍏 Select 1 to 5 foods
- 🏆 High Score / Leaderboard system
- 👤 Custom player name
- 🔁 Retry and Menu screen
- ⌨️ WASD + Arrow key support
- ⏸️ Pause / Resume with Space

---

## 🕹️ Controls

| Key | Action |
|-----|--------|
| ⬆️⬇️⬅️➡️ | Move |
| W A S D | Move |
| Enter | Start Game |
| Space | Pause / Resume |
